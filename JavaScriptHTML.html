Learning JavaScript

Phase 1: JavaScript Fundamentals

1. Introduction to JavaScript

JavaScript is a programming language that runs in the client’s browser and allows you to create interactive web pages.

console.log("Hello, world!");

Resources: MDN Web Docs: JavaScript, Eloquent JavaScript.

2. Basic Syntax

JavaScript syntax is the set of rules that defines how to write code.

let name = "John";
const PI = 3.14159;
let isAdult = age >= 18;

Resources: JavaScript Info: Basics, You Don’t Know JS (Book Series).

3. Control Structures

Control structures in JavaScript are the tools that allow you to control the program's flow.

if (age >= 18) {
    console.log("You are an adult");
} else {
    console.log("You are a minor");
}

for (let i = 0; i < 5; i++) {
    console.log(i);
}

Resources: MDN Web Docs: Control Flow, JavaScript: The Definitive Guide.

4. Functions

Functions are blocks of code that perform a specific task and can be reused in different parts of the program.

function sum(a, b) {
    return a + b;
}

let result = sum(5, 3);
console.log(result);

Resources: Eloquent JavaScript: Functions, JavaScript Info: Functions.

Phase 2: DOM Manipulation

5. Introduction to the DOM

The DOM (Document Object Model) is a structured representation of the HTML document as a tree of nodes.

const header = document.getElementById('myHeader');
console.log(header.textContent);

Resources: MDN Web Docs: Document Object Model (DOM), JavaScript and JQuery: Interactive Front-End Web Development.

6. DOM Manipulation

Manipulating the DOM involves changing the structure, style, or content of HTML elements through JavaScript.

const paragraph = document.querySelector('p');
paragraph.textContent = 'This is new text';
paragraph.style.color = 'blue';

Resources: MDN Web Docs: Manipulating Documents, JavaScript & JQuery: Interactive Front-End Web Development.

7. Events

Events in JavaScript are actions or occurrences that happen in the browser, such as a user click or a page load.

const button = document.querySelector('button');
button.addEventListener('click', function() {
    alert('Button clicked!');
});

Resources: MDN Web Docs: Introduction to events, JavaScript & JQuery: Events.

Phase 3: Diving Deeper into JavaScript

8. Advanced Functions

Advanced functions in JavaScript include concepts like closures, callbacks, and promises.

function createCounter() {
    let counter = 0;
    return function() {
        counter++;
        return counter;
    };
}

const counter1 = createCounter();
console.log(counter1()); // 1
console.log(counter1()); // 2

Resources: You Don’t Know JS: Scope & Closures, MDN Web Docs: Promises.

9. Objects and Arrays

Objects are collections of key-value pairs, while arrays are ordered lists of values.

let person = {
    name: "John",
    age: 30,
    greet: function() {
        console.log("Hello, " + this.name);
    }
};
person.greet();

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]);

Resources: MDN Web Docs: Working with objects, MDN Web Docs: Arrays.

10. Object-Oriented Programming (OOP)

OOP in JavaScript allows you to create object models with attributes and methods, facilitating code reuse and organization.

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    greet() {
        console.log(`Hello, my name is ${this.name}`);
    }
}

const john = new Person("John", 30);
john.greet();

Resources: MDN Web Docs: Object-oriented programming, JavaScript: The Good Parts.
