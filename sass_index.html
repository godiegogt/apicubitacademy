Using Sass with Node.js

This guide will help you set up and use Sass in a Node.js project. Below are the detailed steps:

Step 1: Set Up a Node.js Project

Initialize a new Node.js project:

mkdir my-sass-project
cd my-sass-project
npm init -y

This creates a new directory, navigates into it, and initializes a package.json file.

Step 2: Install Sass

Install Sass as a development dependency:

npm install sass --save-dev

Step 3: Create Sass Files

Create a directory for your Sass files and add your .scss files:

mkdir scss
touch scss/styles.scss

Example content of styles.scss:

$primary-color: #3498db;

body {
    font-family: Arial, sans-serif;
    color: $primary-color;
}

h1 {
    color: darken($primary-color, 10%);
}

Step 4: Compile Sass to CSS

Add a script to your package.json for compiling Sass:

"scripts": {
  "sass": "sass scss:css --style compressed"
}

Then, run the following command to compile your Sass files:

npm run sass

Step 5: Watch for Changes (Optional)

Set up Sass to watch your files for changes:

"scripts": {
  "sass": "sass --watch scss:css --style compressed"
}

Run the watch command to automatically compile your files upon changes:

npm run sass

Step 6: Use Compiled CSS in Your Project

Link the compiled CSS file in your HTML:

<link rel="stylesheet" href="css/styles.css">

Step 7: Optional - Use with Build Tools (e.g., Webpack, Gulp)

Integrate Sass with your build tools if necessary. For example, using Webpack:

npm install sass-loader css-loader style-loader --save-dev

Configure Webpack to use the loaders:

module.exports = {
  module: {
    rules: [
      {
        test: /\.scss$/,
        use: [
          'style-loader',
          'css-loader',
          'sass-loader'
        ],
      },
    ],
  },
};
